GLDraw Controls
===============

These are the default controls for the editor.


Tool Shortcuts
--------------

* Select: V
* Hand / Pan: H
* Line: L
* Rectangle: R
* Ellipse: E


File Shortcuts
--------------

* New document: Ctrl+N
* Open: Ctrl+O
* Save: Ctrl+S


Edit Shortcuts
--------------

* Undo: Ctrl+Z
* Redo: Ctrl+Y or Ctrl+Shift+Z
* Cut: Ctrl+X
* Copy: Ctrl+C
* Paste: Ctrl+V
* Select all: Ctrl+A


View Shortcuts
--------------

* Zoom in: Ctrl+Plus
* Zoom out: Ctrl+Minus
* Zoom to fit: Ctrl+0


Pointer Behavior
----------------

* Shape tools create objects by left-dragging on the canvas.
* The select tool uses click-to-select.
* Shift+Click toggles selection membership.
* Dragging selected objects moves them.
* The hand tool pans the canvas with left-drag.
* The mouse wheel zooms at the cursor.
